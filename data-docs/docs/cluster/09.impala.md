# Impala

## Build

```shell
export PATH=$PATH:$JAVA_HOME/bin
export IMPALA_HOME=`pwd`
export USE_APACHE_COMPONENTS=true
export SKIP_PYTHON_DOWNLOAD=true
export IMPALA_MAVEN_OPTIONS="-Dmaven.test.skip=true -DskipTests"
export IMPALA_BUILD_THREADS=2

sudo apt-get install ninja-build
./buildall.sh -release -noclean -ninja -skiptests -notests -package

sudo apt-get install openjdk-8-jre-headless krb5-user

bin/impala.sh start statestored
bin/impala.sh start catalogd
bin/impala.sh start admissiond
bin/impala.sh start impalad
```
